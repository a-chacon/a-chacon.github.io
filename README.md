# Mysite

This is my personal site, you can visit it at [a-chacon.com](https://a-chacon.com).

## How I build it?

Using:

* [Jekyll](http://jekyllrb.com) for build the site.
* [Tailwind v3](https://tailwindcss.com/) for css.
* [Stimulus](https://stimulus.hotwired.dev/) for js.

And that is all. If you like you can use it like a template forking this repo or cloning.
