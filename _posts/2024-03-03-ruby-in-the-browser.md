WebAssembly (abreviado como Wasm) es un formato de instrucciones binarias que estan hechas para ser leidas por maquinas virtuales stack-based (realiza las operaciones utilizando pilas).

Este formato se creó con el propósito de servir como un objetivo de compilación portátil para diversos lenguajes de programación. ¿Qué significa esto? Básicamente, permite que el código escrito en diferentes lenguajes se compile en un formato común que puede ser ejecutado en la web, tanto en el lado del cliente como en el servidor.
